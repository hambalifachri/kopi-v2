ℹ️ Pusat Informasi

Apa itu Jasdor?

Jasdor (Jasa Direct Order) adalah layanan pesan langsung dari kami. Lewat Jasdor, kamu bisa menikmati kopi dan menu favoritmu dengan harga spesial yang jauh lebih hemat dibandingkan harga normal di outlet. Jajan enak, kantong tetap aman!

Tutorial Cara Order

  1. Pilih menu favoritmu, lalu klik tombol Tambah.
  2. Untuk minuman, sesuaikan rasamu (pilihan es, gula, ukuran, dll). Untuk makanan & promo akan langsung masuk keranjang.
  3. Klik tombol keranjang melayang di bagian bawah layar, periksa pesanan, lalu klik Selesai, Bayar QRIS.
  4. Isi biodata pengiriman dan lakukan pembayaran dengan Scan QRIS yang tampil di layar.
  5. Upload bukti pembayaranmu, lalu klik tombol kirim. Detail pesanan otomatis diteruskan ke WhatsApp Admin!

Install Aplikasi di iPhone

Jika kamu menggunakan iPhone atau iPad, iOS tidak menyediakan tombol install otomatis.

  1. Buka menu Share (kotak dengan panah ke atas).
  2. Pilih Add to Home Screen.
  3. Ikon aplikasi akan tampil di layar utama.

Untuk Android, tombol install akan memicu prompt PWA biasa jika tersedia.
